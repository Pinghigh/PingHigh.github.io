Tikogasa Kernel

本内核由于 ZFS-builtin 违反许可证，所以只提供构建脚本，不分发二进制包。

仓库中的 config 不是默认优化而是我自己写的，精简掉了很多模块（比如 XFS，INTEL 显卡，除了 Realtek 以外的所有声卡等），所以你需要自己搞一份 config。

实际上优化 config 的过程是在 PKGBUILD 里写好了的，所以你把 Arch 官方的配置拿来 build 也没问题。

添加了 cjktty 补丁。

编译时添加了特异化编译，所以你需要在 config 和 PKGBUILD 里找到 ZEN2 znver2 相关字样，换成你自己的架构。
