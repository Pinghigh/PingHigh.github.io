gcc 编译提示 编译器内部错误：File size limit exceeded signal terminated program as

编译的时候爆内存了

可能是你用了形如

的语句进行初始化。
这个语句虽然只会给你的 \(a_0, a_1\) 赋值，但是在编译预处理和程序运行的过程时，它都会将数组全部展开再进行赋值/初始化
